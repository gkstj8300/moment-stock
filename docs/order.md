# [UI/UX 디자인 요청] 오늘의집 스타일의 이커머스 메인 페이지 구현

디자이너, 프론트엔드 페르소나에게
대한민국의 대표적인 라이프스타일 플랫폼 '오늘의집(ohou.se)'의 디자인 시스템(TDS)과 사용자 경험을 분석하여, 
아래 명세에 따라 현대적이고 일관성 있는 이커머스 메인 페이지를 구현해 주세요.

---

## 1. 디자인 철학 및 원칙 (Design Principles)
* [cite_start]**사용자 중심(Customer Centric):** 정보 소외가 없도록 어려운 용어 대신 이해하기 쉬운 UX 라이팅을 적용할 것[cite: 322, 413].
* [cite_start]**일관된 시스템(TDS):** 모든 컴포넌트는 재사용 가능한 시스템(Design System) 기반으로 설계하여 사용자에게 통일된 경험을 줄 것[cite: 428, 430].
* [cite_start]**접근성(Accessibility):** 모든 사용자가 신체적 조건에 관계없이 이용할 수 있도록 WCAG 2.0 및 KWCAG 2.1 지침을 준수할 것[cite: 159, 162].

## 2. 레이아웃 및 그리드 (Layout & Grid)
* **콘텐츠 폭:** 중앙 정렬된 Max-width 1200px 구조를 기본으로 함.
* **시각적 위계:** 상단 GNB -> 메인 비주얼 배너 -> 카테고리 퀵 메뉴(아이콘+텍스트) -> 큐레이션 상품 그리드 순서.
* **반응형:** 모바일 환경에서는 카테고리 메뉴가 횡스크롤(Horizontal Scroll)로 동작하고, 상품 카드는 2열 배치를 기본으로 할 것.

## 3. 컴포넌트 상세 가이드 (Component Specification)
### A. 상품 카드 (Product Card)
* **구조:** 1:1 비율의 라운드 코너(12px) 이미지 + 브랜드명 + 상품명(최대 2줄) + 할인율 및 가격 + 리뷰(별점 및 개수).
* [cite_start]**배지(Badge):** 이미지 좌상단에 '특가', '단독' 등의 배지를 명확히 표시[cite: 441].
* **상태:** 마우스 호버 시 이미지가 미세하게 확대되는 인터랙션 포함.

### B. 컬러 및 타이포그래피
* **Primary Color:** 신뢰감을 주는 스카이블루(#35C5F0)를 포인트 컬러로 활용.
* [cite_start]**Semantic Color:** 다크 모드 대응이 가능하도록 'grey01' 같은 명도 기준이 아닌 'background', 'surface' 등의 의미론적 네이밍 규칙을 적용할 것[cite: 464, 501].
* [cite_start]**Contrast:** 텍스트와 배경의 명도 대비는 최소 4.5:1 이상(AA 등급)을 유지하여 가독성을 확보할 것[cite: 186, 197].

### C. 접근성 요소 (Accessibility)
* [cite_start]**대체 텍스트:** 이미지에는 `alt` 속성을 상세히 작성하고, 상호작용 요소는 스크린 리더가 논리적인 순서로 읽을 수 있도록 설계할 것[cite: 208, 212].
* [cite_start]**초점(Focus):** 키보드 탭 이동 시 현재 위치를 시각적으로 명확히 구별할 수 있어야 함[cite: 211].
* [cite_start]**그룹화:** 카드나 리스트 요소는 개별 항목이 아닌 컴포넌트 단위로 단일 초점이 맺히도록 그룹화할 것[cite: 217].

## 4. 기술 스택 및 출력 요구사항
* **Stack:** React, Tailwind CSS, Lucide-react(아이콘).
* **출력:** 1. 디자인 시스템의 핵심 가이드(Color, Typo) 요약 설명.
    2. 위 명세를 모두 충족하는 단일 파일의 완성된 코드 구현.
