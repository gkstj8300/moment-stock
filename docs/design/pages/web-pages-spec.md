## 2. 공통 레이아웃

모든 페이지는 아래 공통 구조를 따른다.

```
+----------------------------------------------------------+
|  [GNB] 로고 좌측 / 네비 중앙 / 장바구니·로그인 우측       |
|  높이: 56px, border-bottom: 1px #e5e7eb                   |
|  max-width: 1120px, margin: 0 auto                       |
+----------------------------------------------------------+
|                                                          |
|  [Page Content]                                          |
|  max-width: 1120px, margin: 0 auto                       |
|  padding: spacing-8(32px) 상하, spacing-6(24px) 좌우      |
|  모바일: padding spacing-4(16px) 좌우                     |
|                                                          |
+----------------------------------------------------------+
```

### 공통 반응형 브레이크포인트

| 구간 | 범위 | 열 수 | 좌우 패딩 |
|------|------|-------|-----------|
| 모바일 | 360px ~ 767px | 1열 | `spacing-4` (16px) |
| 태블릿 | 768px ~ 1279px | 2열 | `spacing-6` (24px) |
| 데스크톱 | 1280px ~ | 3열 | `spacing-6` (24px) |

## 3. 홈 (`/`)

### 3.1. 페이지 목적

타임세일 긴박감을 전달하고, 전체 상품을 그리드로 탐색할 수 있는 진입점.

### 3.2. 와이어프레임

```
+----------------------------------------------------------+
|                      [GNB]                                |
+----------------------------------------------------------+
| spacing-8 (32px)                                          |
| +------------------------------------------------------+ |
| |  [타임세일 배너]                                       | |
| |  gradient: from-red-500(#ef4444) to-orange-500(#f97316)| |
| |  border-radius: 12px, padding: spacing-6(24px)        | |
| |                                                       | |
| |  "타임 세일 진행 중"  ← Display 36px, Bold, White      | |
| |  "지금 바로 특가 상품을 만나보세요" ← Body 16px, White  | |
| |  spacing-4 (16px)                                     | |
| |  "세일 종료까지" ← Caption 12px, White 75% opacity     | |
| |  [ 02:34:56 ] ← Timer 24px, Bold, tabular-nums, White | |
| +------------------------------------------------------+ |
| spacing-6 (24px)                                          |
| "전체 상품" ← H2 20px, Semibold, Gray 900                |
| spacing-4 (16px)                                          |
| +----------------+ +----------------+ +----------------+ |
| | [상품 카드]     | | [상품 카드]     | | [상품 카드]     | |
| | 이미지 aspect   | | 이미지 aspect   | | 이미지 aspect   | |
| | 4:3, 12px rad   | | 4:3             | | 4:3             | |
| | spacing-3(12px) | |                 | |                 | |
| | 상품명 Body 16px| |                 | |                 | |
| | 가격 Price 20px | |                 | |                 | |
| | 재고 게이지     | |                 | |                 | |
| +----------------+ +----------------+ +----------------+ |
|    gap: spacing-4 (16px)                                  |
+----------------------------------------------------------+
```

### 3.3. 섹션별 상세

#### 타임세일 배너

| 속성 | 값 |
|------|-----|
| 배경 | `linear-gradient(to right, #ef4444, #f97316)` |
| 모서리 | `borderRadius.lg` (12px) |
| 패딩 | `spacing-6` (24px), 태블릿 이상 `spacing-8` (32px) |
| 제목 | Display 36px, Bold 700, White |
| 부제 | Body 16px, Regular 400, White, opacity 0.9 |
| 타이머 라벨 | Caption 12px, White, opacity 0.75 |
| 타이머 값 | Timer 24px, Bold 700, `tabular-nums`, White |
| 제목~부제 간격 | `spacing-1` (4px) |
| 부제~타이머 간격 | `spacing-4` (16px) |

#### 상품 그리드

| 속성 | 값 |
|------|-----|
| 섹션 제목 | H2 20px, Semibold 600, Gray 900 (`#111827`) |
| 제목~그리드 간격 | `spacing-4` (16px) |
| 그리드 거터 | `spacing-4` (16px) |
| 카드 스타일 | 디자인 파운데이션 7.2 카드 규격 적용 |

### 3.4. 반응형 동작

| 구간 | 동작 |
|------|------|
| 모바일 | 배너 padding `spacing-6`(24px), 그리드 1열, 카드 full-width |
| 태블릿 | 배너 padding `spacing-8`(32px), 그리드 2열 |
| 데스크톱 | 배너 padding `spacing-8`(32px), 그리드 3열 |

### 3.5. 상태별 UI

| 상태 | 표현 |
|------|------|
| 로딩 | 배너: 그래디언트 배경 유지 + 텍스트 스켈레톤. 그리드: 카드 형태 스켈레톤 6개 (pulse) |
| 에러 | 배너 정상 표시. 그리드 영역에 에러 알림: 배경 `#fef2f2`, 텍스트 `#dc2626`, 12px radius |
| 빈 상태 | 배너 정상 표시. "등록된 상품이 없어요" Body 16px, Gray 500, 중앙 정렬, py `spacing-8` |

### 3.6. 인터랙션

| 트리거 | 동작 |
|--------|------|
| 카드 호버 | `shadow.md` 전환 + 이미지 `scale(1.05)`, 250ms ease-out |
| 카드 클릭 | `/products/[id]`로 이동 |
| 카운트다운 1분 미만 | 타이머 색상 `#fef2f2` 배경으로 전환 + `scale(1.05)`, 250ms |
| `prefers-reduced-motion` | 모든 transition, scale 비활성화 |

### 3.7. UX Writing

| 위치 | 텍스트 |
|------|--------|
| 배너 제목 | "타임 세일 진행 중" |
| 배너 부제 | "지금 바로 특가 상품을 만나보세요" |
| 타이머 라벨 | "세일 종료까지" |
| 섹션 제목 | "전체 상품" |
| 로딩 | "잠시만요..." |
| 에러 | "상품을 불러오지 못했어요. 잠시 후 다시 시도해주세요." |

### 3.8. 접근성

| 항목 | 구현 |
|------|------|
| 배너 | `aria-label="타임 세일 배너"`, `<section>` |
| 타이머 | `aria-live="polite"`, `role="timer"` |
| 상품 그리드 | `aria-label="상품 목록"`, `<section>` |
| 로딩 | `role="status"` |
| 에러 | `role="alert"` |
| 포커스 순서 | 배너 → 섹션 제목 → 카드 순서 (DOM 순) |
| 카드 포커스 | `focus-visible` 2px Blue 600 outline |

## 5. 상품 상세 (`/products/[id]`)

### 5.1. 페이지 목적

상품의 상세 정보와 실시간 재고를 확인하고 장바구니에 담을 수 있는 페이지.

### 5.2. 와이어프레임

```
+----------------------------------------------------------+
|                      [GNB]                                |
+----------------------------------------------------------+
| spacing-8 (32px)                                          |
| +------------------------+  spacing-8  +----------------+ |
| | [상품 이미지]           |   (32px)    | [상품 정보]     | |
| | aspect-square           |            |                 | |
| | bg: Gray 100            |            | 상품명           | |
| | border-radius: 12px     |            | H1 24px Bold    | |
| | object-cover            |            | spacing-2 (8px) | |
| |                         |            | 설명 Body 16px  | |
| |                         |            | Gray 600        | |
| |                         |            | spacing-6(24px) | |
| |                         |            | ₩15,000         | |
| |                         |            | Price Lg 28px   | |
| |                         |            | Bold, Gray 900  | |
| |                         |            | spacing-6(24px) | |
| |                         |            | [재고 표시]     | |
| |                         |            | 게이지 바 6px   | |
| |                         |            | + 뱃지 + 수량   | |
| |                         |            | spacing-6(24px) | |
| |                         |            | [장바구니 담기] | |
| |                         |            | Primary 버튼    | |
| |                         |            | w-full, 44px h  | |
| +------------------------+             +----------------+ |
+----------------------------------------------------------+
```

### 5.3. 섹션별 상세

#### 이미지 영역 (좌측)

| 속성 | 값 |
|------|-----|
| 비율 | `aspect-square` (1:1) |
| 배경 | Gray 100 (`#f3f4f6`) |
| 모서리 | 12px |
| 이미지 핏 | `object-cover` |
| 이미지 없을 때 | "No Image" Body 16px, Gray 400, 중앙 정렬 |

#### 상품 정보 영역 (우측)

| 요소 | 타이포그래피 | 색상 | 하단 간격 |
|------|-------------|------|-----------|
| 상품명 | H1 24px, Bold 700 | Gray 900 (`#111827`) | — |
| 설명 | Body 16px, Regular 400 | Gray 600 (`#4b5563`) | `spacing-6` (24px) |
| 가격 | Price Large 28px, Bold 700 | Gray 900 | `spacing-6` (24px) |
| 재고 표시 | 게이지 바 + 텍스트 + 뱃지 | 상태별 컬러 | `spacing-6` (24px) |
| CTA 버튼 | Body 16px, Semibold 600 | White on Blue 600 | — |

#### 재고 표시 컴포넌트

| 요소 | 스펙 |
|------|------|
| 게이지 바 배경 | Gray 200 (`#e5e7eb`), 높이 8px, `borderRadius.full` |
| 게이지 바 채움 | 상태별 컬러 (Green/Yellow/Orange/Red/Gray), transition 250ms |
| 수량 텍스트 | Body Small 14px, 상태별 컬러 |
| 희소성 뱃지 | Caption 12px, 상태별 뱃지 (디자인 파운데이션 3.2 참조) |
| 위험 상태 | 게이지 바 pulse 애니메이션, "마지막 N개!" Red 뱃지 + 🔥 |

#### CTA 버튼

| 상태 | 배경 | 텍스트 | 동작 |
|------|------|--------|------|
| 기본 | Blue 600 (`#2563eb`) | "장바구니에 담기" White | 호버 Blue 700 |
| 품절 | Gray 300 (`#d1d5db`) | "품절" Gray 500 | `disabled`, `cursor-not-allowed` |
| 오프라인 | Gray 300 | "오프라인" Gray 500 | `disabled` |
| 클릭 중 | Blue 600 | `scale(0.98)` | 150ms |

### 5.4. 반응형 동작

| 구간 | 동작 |
|------|------|
| 모바일 | 1열 세로 배치 (이미지 위, 정보 아래). 이미지 full-width. 간격 `spacing-6`(24px) |
| 태블릿 이상 | 2열 가로 배치 (`grid-cols-2`). 이미지:정보 = 1:1. 간격 `spacing-8`(32px) |

### 5.5. 상태별 UI

| 상태 | 표현 |
|------|------|
| 로딩 | 2열 레이아웃 유지. 좌: 이미지 스켈레톤 (Gray 200, pulse). 우: 텍스트 스켈레톤 4줄 |
| 에러/404 | "상품을 찾을 수 없습니다." 배경 `#fef2f2`, 텍스트 `#dc2626`, 중앙 정렬 |
| 품절 전환 | 이미지 `grayscale(1)` + "품절" 뱃지 페이드인 350ms |

### 5.6. 인터랙션

| 트리거 | 동작 |
|--------|------|
| 장바구니 담기 | 버튼 `scale(0.98)` 150ms → 토스트 "장바구니에 담았어요" (Success, 3초) |
| 실시간 재고 변동 | 게이지 바 너비 transition 250ms ease-out, 수량 숫자 transition 250ms |
| 재고 0 전환 | 게이지 Gray, 이미지 grayscale, "품절" 뱃지 표시, 버튼 disabled |
| `prefers-reduced-motion` | pulse, scale, grayscale transition 비활성화 |

### 5.7. UX Writing

| 위치 | 텍스트 |
|------|--------|
| CTA 버튼 | "장바구니에 담기" |
| CTA 품절 | "품절" |
| CTA 오프라인 | "오프라인" |
| 토스트 성공 | "장바구니에 담았어요" |
| 로딩 | "상품 정보를 불러오는 중..." |
| 에러 | "상품을 찾을 수 없습니다." |
| 재고 부족 뱃지 | "곧 품절" |
| 재고 위험 뱃지 | "마지막 {N}개!" |
| 재고 품절 뱃지 | "품절" |

### 5.8. 접근성

| 항목 | 구현 |
|------|------|
| 이미지 | `alt="{상품명}"` |
| 재고 게이지 | `role="progressbar"`, `aria-valuenow`, `aria-valuemin=0`, `aria-valuemax=100`, `aria-label="재고 {N}%"` |
| 재고 수량 | `aria-live="polite"` (실시간 변동 반영) |
| CTA 버튼 | `aria-label="{상품명} 장바구니에 담기"` (품절 시 `aria-label="품절된 상품입니다"`) |
| 포커스 순서 | 이미지 → 상품명 → 가격 → 재고 → CTA 버튼 |
| 색상 무관 | 재고 상태를 컬러 + 아이콘 + 텍스트로 중복 표현 |

## 6. 장바구니 (`/cart`)

### 6.1. 페이지 목적

담은 상품을 확인, 수량 조절, 삭제하고 결제로 진행하는 페이지.

### 6.2. 와이어프레임

```
+----------------------------------------------------------+
|                      [GNB]                                |
+----------------------------------------------------------+
| spacing-8 (32px)                                          |
| +------------------------------------------------------+ |
| | "장바구니" H1 24px Bold        "전체 삭제" →          | |
| |                            Body Small 14px Red 500   | |
| +------------------------------------------------------+ |
| spacing-6 (24px)                                          |
| +------------------------------------------------------+ |
| | [장바구니 아이템 리스트]  border 1px, 12px radius      | |
```

| 에러 | "대기열 등록에 실패했습니다. 다시 시도해주세요." |
| 유효하지 않은 접근 | "유효하지 않은 접근입니다." |

### 12.8. 접근성

| 항목 | 구현 |
|------|------|
| 순번 | `aria-live="polite"` (5초 간격 업데이트 반영) |
| 등록 중 | `role="status"` |
| 에러 | `role="alert"` |
| 입장 허가 | `aria-live="assertive"` (즉시 스크린 리더에 전달) |
| 자동 리다이렉트 | 리다이렉트 전 스크린 리더에 "타임 세일 페이지로 이동합니다" 전달 |
| 포커스 순서 | 제목 → 순번 카드 → 안내 텍스트 |
