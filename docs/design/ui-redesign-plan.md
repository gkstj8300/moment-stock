## 2. 디자인 방향

### 2.1. 핵심 변경 — 포인트 컬러

| 항목 | 기존 | 변경 |
|------|------|------|
| **Primary** | `#2563eb` (Blue 600) | `#fa2454` (moment-stock Red) |
| **Primary Hover** | `#1d4ed8` | `#e01e4a` |
| **Primary Pressed** | `#1e40af` | `#c91a40` |
| **CTA 버튼** | Blue 계열 | `#fa2454` Red 계열 |
| **강조 뱃지** | Blue/Red 혼용 | `#fa2454` 통일 |
| **GNB 액센트** | Red 텍스트 | `#fa2454` 배경 버튼 |

### 2.2. 레퍼런스 분석 (오늘의집 8장)

| 이미지 | 추출 패턴 | moment-stock 적용 |
|--------|-----------|-------------------|
| **메인페이지** | 상단 프로모션 배너 (노란색) + 카테고리 아이콘 그리드 + 상품 카드 리스트 + Footer | 타임세일 배너 + 카테고리(없음→상품 필터) + 상품 그리드 + Footer |
| **카테고리 1depth** | 좌측 사이드바 카테고리 목록 + 우측 배너 + 상품 그리드 | moment-stock은 카테고리 없음 → 필터/정렬 드롭다운으로 대체 |
| **카테고리 2depth** | 상단 배너 이미지 + 탭(베스트/특가/신상) + 상품 그리드 | 타임세일 페이지에 탭 네비게이션 추가 |
| **카테고리 3depth** | 좌측 사이드바 + SPECIAL 프로모션 배너 + BEST 상품 5개 + 추가 상품 그리드 | 상품 목록에 "인기 상품" 섹션 추가 |
| **카테고리 4depth** | 좌측 사이드바 + 상단 큰 이미지 상품 + 필터 탭(추천/인기/낮은가격/높은가격/리뷰) + 상품 그리드 | 정렬 옵션 추가 |
| **회원가입** | 로고 + SNS 간편가입 3종 + 이메일/비번/비번확인/닉네임 + 약관동의 체크박스 + CTA 풀와이드 | 인증 폼 리디자인: border-bottom 입력 필드, 풀와이드 CTA |
| **로그인** | 중앙 정렬 로고 + 카드형 이메일/비번 입력 + 풀와이드 CTA 버튼 (하늘색→#fa2454) + SNS 로그인 | 로그인 폼 리디자인: 카드형 입력, 풀와이드 `#fa2454` CTA |
| **커뮤니티** | 상단 배너 + 카테고리 아이콘 스크롤 + 사진/비디오 콘텐츠 카드 + "오늘의 추천" 섹션 | 현재 해당 없음 → 향후 확장 시 참조 |

### 2.3. UI 스타일 가이드 변경

| 항목 | 기존 | 변경 (레퍼런스 기반) |
|------|------|---------------------|
| **GNB** | backdrop-blur 반투명 | 순백 배경 `#fff` + border-bottom `#f0f0f0` |
| **GNB 높이** | 56px | 동일 유지 |
| **GNB 구조** | 로고 좌 + 링크 우 | 로고 좌 + 검색바 중앙 + 아이콘(장바구니/로그인) 우 |
| **카드 테두리** | 1px `#e5e7eb` | 테두리 없음, `shadow-sm` 기본 + hover `shadow-md` |
| **카드 모서리** | 12px | 8px (더 타이트) |
| **입력 필드** | rounded-xl + border | border-bottom only (회원가입) / 카드형 (로그인) |
| **CTA 버튼** | Blue 600, rounded-xl | `#fa2454`, 8px radius, 풀와이드 |
| **할인 뱃지** | Red 배경 소형 | `#fa2454` 배경, Bold, 가격 왼쪽 |
| **가격 표시** | 단일 표시 | 할인율 + 할인가(Bold, 크게) + 원가(line-through, 작게) |
| **별점/리뷰** | 없음 | 별점 + 리뷰 수 표시 (추후) |
| **배송 뱃지** | 없음 | "무료배송" 뱃지 (Gray) |
| **페이지 배경** | `#f9fafb` (Gray 50) | `#ffffff` (순백) |
| **Footer** | 없음 | 추가 — 고객센터, 사업자정보, 카피라이트 |

## 4. 페이지별 리디자인 계획

### R1. GNB (공통 헤더)

```
┌──────────────────────────────────────────────────────────────┐
│ [Logo]      [──────── 검색 ────────]  🔔  🛒(N)  👤        │
│ moment-stock   placeholder           알림 장바구니 로그인    │
├──────────────────────────────────────────────────────────────┤
│ [타임세일]  [베스트]  [신상품]  [이벤트]                      │  서브 네비
└──────────────────────────────────────────────────────────────┘
  bg: #fff, border-bottom: 1px #f0f0f0
  높이: 1단(56px) + 서브네비(44px) = 100px
```

## 6. 변경되지 않는 것

| 항목 | 사유 |
|------|------|
| 재고 게이지 바 5단계 컬러 (green/yellow/orange/red/gray) | 재고 상태는 의미론적 컬러 유지 (포인트 컬러와 무관) |
| 접근성 기준 (4.5:1 대비, 44px 터치, aria) | 접근성은 유지 |
| 컴포넌트 아토믹 구조 (atoms/molecules/organisms) | 구조는 유지, 스타일만 변경 |
| @repo/shared 비즈니스 로직 | UI만 변경, 로직 불변 |
| 모바일 앱 (Phase 완료 후 별도 진행) | 웹 먼저 |
