## 7. 컴포넌트 기본 스타일

### 7.1. 버튼

| Variant | 배경 | 텍스트 | 테두리 | 높이 |
|---------|------|--------|--------|------|
| Primary | Blue 600 | White | 없음 | 44px (md) / 48px (lg) |
| Secondary | White | Gray 900 | Gray 300 | 44px |
| Ghost | 투명 | Gray 600 | 없음 | 44px |
| Danger | Red 500 | White | 없음 | 44px |
| Disabled | Gray 300 | Gray 500 | 없음 | 44px |

- 모서리: `borderRadius.lg` (12px)
- 호버: 한 단계 어두운 색
- 트랜지션: `animation.duration.normal` (250ms)

### 7.2. 카드

| 속성 | 값 |
|------|-----|
| 배경 | White |
| 테두리 | `border.default` (#e5e7eb) 1px |
| 모서리 | `borderRadius.lg` (12px) |
| 그림자 | 없음 (호버 시 `shadow.md`) |
| 패딩 | `spacing-4` (16px) |
| 호버 | `shadow.md` + 이미지 scale 1.05 |

### 7.3. 입력 필드

| 속성 | 값 |
|------|-----|
| 높이 | 44px (min-height) |
| 배경 | White |
| 테두리 | `border.default` 1px → 포커스 시 Blue 600 2px |
| 모서리 | `borderRadius.lg` (12px) |
| 패딩 | 12px 16px |
| 라벨 | Body Small (14px), Medium, Gray 900 |
| 에러 | Body Small (14px), Red 600 |

### 7.4. 뱃지

| Variant | 배경 | 텍스트 |
|---------|------|--------|
| Default | Gray 100 | Gray 700 |
| Success | Green 100 | Green 700 |
| Warning | Yellow 100 | Yellow 800 |
| Error | Red 100 | Red 700 |
| Info | Blue 100 | Blue 700 |

- 크기: Caption (12px), 패딩 6px 10px
- 모서리: `borderRadius.full` (9999px)

## 8. 인터랙션 패턴

### 8.1. 마이크로인터랙션

| 트리거 | 피드백 | 지속 시간 |
|--------|--------|-----------|
| 장바구니 담기 | 토스트 "장바구니에 담았어요" (Success) | 3초 |
| 구매 완료 | 토스트 "구매 완료!" (Success) | 3초 |
| 구매 실패/품절 | 토스트 에러 메시지 (Error) | 3초 |
| 재고 감소 | 숫자 transition (250ms ease-out) | 250ms |
| 품절 전환 | 카드 grayscale + "품절" 뱃지 페이드인 | 350ms |
| 카운트다운 1분 미만 | 타이머 색상 빨강 전환 + 약간 확대 | 250ms |
| 호버 (카드) | 그림자 + 이미지 확대 | 250ms |
| 버튼 클릭 | 약간의 scale(0.98) | 150ms |

### 8.2. 로딩 상태

| 상태 | 표현 |
|------|------|
| 페이지 로딩 | 스켈레톤 (카드 형태의 회색 블록, pulse 애니메이션) |
| 버튼 로딩 | 텍스트 → "처리 중..." + disabled |
| 데이터 리패치 | 기존 데이터 유지, 백그라운드 갱신 |

### 8.3. 빈 상태

| 상태 | 메시지 | 액션 |
|------|--------|------|
| 장바구니 비었을 때 | "아직 담은 상품이 없어요" | "쇼핑하러 가기" 버튼 |
| 주문 내역 없을 때 | "아직 주문 내역이 없어요" | — |
| 검색 결과 없을 때 | "찾는 상품이 없어요" | — |
| 타임 세일 없을 때 | "지금은 진행 중인 세일이 없어요" | — |

## 9. 접근성 기준

| 항목 | 기준 |
|------|------|
| 명도 대비 | 텍스트/배경 4.5:1 이상 (AA) |
| 터치 타겟 | 최소 44x44px |
| 포커스 | `focus-visible` 2px Blue 600 outline |
| 키보드 | 모든 인터랙티브 요소 Tab 접근 가능 |
| 스크린 리더 | `aria-label` 필수, 동적 영역 `aria-live` |
| 모션 | `prefers-reduced-motion` 시 애니메이션 비활성화 |
| 색상 무관 | 재고 상태를 색상 + 아이콘 + 텍스트로 중복 표현 |
