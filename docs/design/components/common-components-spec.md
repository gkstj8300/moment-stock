## 2. Header (공통 헤더)

### 구조

```
┌─────────────────────────────────────────────────────────┐
│ [Logo]           [타임세일] [상품] [장바구니(N)] [로그인] │  h: 56px
└─────────────────────────────────────────────────────────┘
```

### 상세

| 요소 | 스타일 |
|------|--------|
| 컨테이너 | sticky top-0, bg white/80 + backdrop-blur, border-bottom gray-200, h-56px, z-50 |
| 로고 | 20px Bold, Gray 900, Link → "/" |
| 타임세일 링크 | 14px Semibold, Red 500, hover underline |
| 일반 링크 | 14px Regular, Gray 600, hover underline |
| 장바구니 | 14px Regular + 뱃지(카운트 > 0일 때 Red 500 원형, 12px White) |
| 내부 max-width | 1120px, 좌우 padding spacing-4 |

### 반응형

| 화면 | 변화 |
|------|------|
| 모바일 | 로고 + 햄버거 메뉴 (또는 아이콘만 표시) |
| 태블릿+ | 전체 네비게이션 표시 |

## 3. ProductCard (상품 카드)

### 구조 (Compound 패턴)

```
┌──────────────────────┐
│  ┌──────────────────┐ │
│  │                  │ │  Image: aspect-square
│  │     이미지        │ │  hover → scale 1.05
│  │                  │ │
│  └──────────────────┘ │
│  ┌──────────────────┐ │
│  │ 상품명            │ │  14px Medium, truncate
│  │ 89,000원          │ │  20px Bold
│  │ ■■■■■□□□ 42개 남음 │ │  StockDisplay
│  └──────────────────┘ │  padding: spacing-4
└──────────────────────┘   border: gray-200 1px, radius: 12px
```

### 상태별

| 상태 | 변화 |
|------|------|
| 기본 | 위 구조 |
| 호버 | shadow-md + 이미지 scale(1.05), 250ms ease-out |
| 품절 | 이미지 grayscale(50%) + "품절" 뱃지 오버레이, 가격 gray-400 |
| 로딩 | 스켈레톤: 이미지 영역 + 텍스트 3줄 pulse |

### 할인 적용 시

```
│ [30%] 62,300원           │  Badge(Error) + Price(Red 600, 20px Bold)
│ ̶8̶9̶,̶0̶0̶0̶원̶               │  원가 14px line-through Gray 400
```

## 4. StockDisplay (재고 표시)

### 구조

```
┌─────────────────────────────────────┐
│ 42개 남음              [곧 품절]     │  14px Semibold(상태색) + Badge
│ ■■■■■■■■■■■■□□□□□□□□□□□□□□□□□□□□□□ │  게이지 바: h-6px, radius-full
└─────────────────────────────────────┘
```

### 상태별 스타일

| 상태 | 텍스트 색 | 게이지 색 | 뱃지 |
|------|-----------|-----------|------|
| 여유 (>50%) | Green 600 | Green 500 | — |
| 보통 (20~50%) | Yellow 600 | Yellow 500 | — |
| 부족 (5~20%) | Orange 600 | Orange 500 | "곧 품절" (Orange bg) |
| 위험 (1~5개) | Red 600 | Red 500 + pulse | "마지막 N개!" (Red bg, pulse) |
| 품절 (0개) | Gray 500 | Gray 400 | "품절" (Gray bg) |

### 접근성

- `role="progressbar"`, `aria-valuenow`, `aria-valuemin=0`, `aria-valuemax=100`
- `aria-label="재고 N% 남음"`
- 뱃지의 pulse 애니메이션: `motion-reduce:animate-none`

## 5. CountdownTimer (카운트다운 타이머)

### 구조

```
  02:34:15    ← 24px Bold, tabular-nums, monospace
```

### 상태별

| 상태 | 스타일 |
|------|--------|
| 1시간 이상 | Gray 900, 24px |
| 10분~1시간 | Gray 900, 24px |
| 1분~10분 | Orange 600, 24px |
| 1분 미만 | Red 500, 28px (약간 확대), 250ms transition |
| 종료 | "세일 종료", 14px Medium, Gray 500 |

### 접근성

- `aria-live="polite"` (매초 변경되지만 polite로 과도한 알림 방지)
- `aria-label="남은 시간 N시간 N분 N초"`
- `<time>` 시맨틱 태그 사용

## 6. Toast (토스트 알림)

### 구조

```
           ┌────────────────────────┐
           │ ✓ 장바구니에 담았어요    │  14px Medium, White
           └────────────────────────┘  fixed bottom-80px center
```

### Variant별

| Variant | 배경 | 아이콘 |
|---------|------|--------|
| Success | Green 600 | ✓ |
| Error | Red 600 | ✕ |
| Info | Gray 800 | ℹ |

### 동작

- 등장: slide-up + fade-in (250ms)
- 지속: 3초
- 퇴장: fade-out (250ms)
- `motion-reduce`: 즉시 표시/숨김 (애니메이션 없음)
- `aria-live="polite"`, `role="status"`

- `role="alert"`, `aria-live="assertive"` (즉시 알림)

## 8. Skeleton (스켈레톤 로딩)

### 사용처별 형태

**상품 카드 스켈레톤:**
```
┌──────────────────────┐
│ ████████████████████ │  aspect-square, Gray 200, pulse
│ ████████████████████ │
│ █████████            │  h-16px, w-60%
│ ████████             │  h-20px, w-50%
│ ██████████████████   │  h-6px, w-100% (게이지)
└──────────────────────┘
```

**상품 상세 스켈레톤:**
```
┌────────────────┐  ┌──────────────────────┐
│ ██████████████ │  │ ████████████         │ h-28px
│ ██████████████ │  │ ████████             │ h-16px
│ ██████████████ │  │ ██████████████████   │ h-32px
│ ██████████████ │  │ ██████████████████   │ h-6px
└────────────────┘  │ [████████████████]   │ h-44px
                    └──────────────────────┘
```

- 색상: Gray 200
- 애니메이션: pulse (opacity 0.5↔1, 2s infinite)
- `motion-reduce`: 애니메이션 없이 정적 Gray 200 블록
- `role="status"`, `aria-label="로딩 중"`
